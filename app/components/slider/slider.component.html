Slider

Sliders allow users to select a value from a range using drag and drop.

Use Case

Use Sliders to help users select a value from a range between a minimum and maximum value. Labels can be added to each side of the slider for clarity. Sliders can be orientated on a horizontal or vertical axis. Consider using an Input if the value needs to be precise.
