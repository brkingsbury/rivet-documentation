Carousel

The Carousel is a way of displaying multiple panels of information non-simultaneously.

Use Case

Use a Carousel in cases where it is important to show multiple similar panels of information in a compact area (e.g. a list of goals).

Implementation

To operate correctly, the carousel requires the advicent-rivet-carousel component to wrap the slide content that will be included in the carousel. Each slide must be marked with the advicentRivetCarouselSlide attribute directive, else it will not be recognized as a slide.
