Links

Links are clickable elements that navigate users to a new page.

Use Case

Use Links when navigating to new pages within the same site or outside of the site.

Implementation

Text links

Link

Links as buttons

Use Case

Use Links as Buttons to draw more attention to the link with styling similar to buttons. These links still direct users to another page.

Implementation

In some instances, links to be more of a call to action but still direct users to another page. In these cases we use links styled similar to buttons. This can be done by applying any of the button classes to an <a> tag.
