Tooltips

Tooltips provide contextual information that help a user understand an element on the page.

Implementation

Tooltip (on mouseover)

Use case

Use Mouseover Tooltips to label or provide contextual information for unfamiliar icons and components.

Tooltip (delayed)

Use case

Tooltips with a delay are useful for tooltips where we don't want it to be shown immediately on hover. Delay is set in milliseconds.

Tooltip (manual trigger)

Use case

Use Manually Triggered Tooltips to call attention to something that has occurred automatically.
