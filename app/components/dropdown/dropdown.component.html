Dropdowns

Dropdowns are inputs that allow users to select an option from a list.

Use Case

Use Dropdowns when there is only one option the user can choose and 5-10 options to choose from. Consider using Radio Buttons if there are less than 5 options or an Input field with autocomplete if there are more than 10 options.

Implementation

Dropdowns currently use the Angular Material dropdown component and directives as their basis.
