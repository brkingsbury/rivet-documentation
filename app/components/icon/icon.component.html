Icon

Implementation

To add an icon, use the rvt-icon element and set the name="" to the name of the icon you want to use.

There are 4 available icon sizes to use, which can be set using size=""

sm

(15px)

md

(20px, default)

lg

(30px)

xl

(40px)

Styling icons

To style an icon (size, color, etc) use the same rules for styling text, such as font-size, color. Icons should only be set to 1 of the 4 available sizes: 15px, 20px, 30px or 40px.

It should be noted that icons from this font cannot contain more than one color. To insert an icon with multiple colors you should insert an SVG image. SVG images are infinitely scalable an can also be styled with color variables if set up correctly, i.e. imported as code.

There are three common color variables that icons will be assigned. They will be called out in annotated mockups which variable to use.

  • Text icon color: var(--rvt-color-text)
  • Neutral Icon color: var(--rvt-color-icon-neutral)
  • Color Icon color: var(--rvt-color-icon-color)
