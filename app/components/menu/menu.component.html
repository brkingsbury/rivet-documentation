Menu / Popover

Added v5.9.4

Menus use Angular Material for their functionality, for more information see Material documentation

Use case

Menus are used to condense content or actions such as filters, and are attached to triggers such as buttons.

Implementation

Add @import "menuStyles"; in your sites main SCSS file in order to get the Rivet styles.

Use trigger.openMenu() and trigger.closeMenu() to conditionally open and close the menu, either from within the menu view or not.

Simple menu

Menus are simply containers that can include any variety of content.

Menu with interactive content

In order to include interactive content within your menu, add $event.stopPropagation() to any click event to prevent the menu from closing by default.

You can put different content into menus


Including inputs, dropdowns, datepickers, and other buttons
