Snackbar

Snackbars are low emphasis notifications that prompt user action or confirm an action has happened.

Use case

Use Snackbars to gently inform users of application actions outside of the userâ€™s control. Snackbar notifications can communicate actions completed successfully, errors, connection problems, and action delays. Snackbars are configured to dismiss manually or automatically with a time limit. Consider using a Dialog for high priority notifications.

Implementation

Snackbars currently use the Angular Material snackbar component and directives as their basis. In order to have more control over the parts within the snackbar, it is advised to use the separate template for the snackbar structure.
For more information on Material snackbars, see Snackbar documentation.
