Dialog

Dialogs are overlays that require users to make or confirm a decision, or step through a process to complete a task.

Use Case

Use Dialogs to notify users of important decisions that need to be confirmed or made. Dialogs are also used to break up large tasks into a few steps. Make sure dialog information and instructions are informative, concise, and clear. Consider using a Snackbar for a less disruptive notification.

Implementation

Dialogs currently use the Angular Material dialog component and directives as their basis. Within the dialogRef, set it to autoFocus: false to prevent the buttons from being focused on open.

Dialog implementation is split into two components. One component is created with the content of the dalog, and the other passes the content component to the MatDialog service.
