Table

Tables use rows and columns to organize data in a dense format.

Use case

Use Tables to display large sets of data using rows and columns. Search, sorting, and filtering features may be added to improve organization.

Implementation

Tables are built off of Angular material. For more documentation see Material tables.
Tables should not use the default material class="mat-elevation-x"

Pagination

Use case

Use Pagination when tables become too long to display on 1 page.

Implementation

Pagination uses the Angular Material pagination component. For more documentation see Material pagination.
