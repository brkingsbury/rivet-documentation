Loading

Loading indicators are used to communicate to the user that a section or page is in the process of loading.

Use Case

Use Loading Indicators when it takes longer than 1 second to load a page or section. Selecting an indicator depends on the loading time and how the action was initiated.

Material Loading

Implementation

For indeterminate loading states, mat-spinner is used along with aria-label="loading" for screen readers.

Loading Section

Use Case

Use Loading Sections when page content is being reloaded or when a new tab is selected.

Implementation

Loading sections are used within larger page sections. To add text, simply include span class="loading-text".

Loading Buttons

Use Case

Use Loading Buttons when a major action or process takes longer than 1 second to complete.

Implementation

Buttons in a loading state need to be disabled and include the class loading to apply the correct styles as well as prevent interaction while loading
