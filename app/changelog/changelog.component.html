Roadmap

Rivet v4

  • Development has begun for Rivet v4 in preparation for the NaviPlan Essentials project. It is intended to supply all the necessary components and styles to make a beta release of the Essentials tool a success.
  • Continued iteration and patching is intended after the initial beta release to provide the best user experience in the Essentials product.

Rivet v3

  • Rivet v3 has been released to support the Guided Retirement project. It is intended to inclued all the components and styles required to make that project a success.
  • v3.5 will continue to receive patch versions as necessary to provide for Guided Retirement bug fixes.
