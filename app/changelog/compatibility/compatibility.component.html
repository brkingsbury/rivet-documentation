Compatibility

This catalogues the various compatibility considerations that arise with Rivet and with branding configurations within Advicent's software suite.

NaviPlan Premium

Version

Rivet is not utilized in NaviPlan Premium, but v3 was used in the 2019 visual update to inform the visual design of the software.

Branding

NaviPlan follows its own branding variable standard. Documentation for branding capabilities can be found here: NaviPlan Branding Documents

NaviPlan Essentials

Version

NaviPlan Essentials has been proven to be compatible with Rivet v4, and will be closely tied to Rivet development for the forseeable future.

Branding

NaviPlan Essentials follows the latest branding variable standards as established in Rivet v4, and can be color-customized through the Auth Server’s ability to apply enterprise branding packages on login.

Guided Retirement

Version

Guided Retirement has been proven to be compatible with Rivet v4. It is recommended that it remain on Rivet v4 unless a major update is performed (e.g. an Angular version update) that would require a Rivet update. It is also possible that future patch updates be applied if bugs caused by Rivet are found.

Branding

Guided Retirement follows the branding variable standards as established in Rivet v4, and can be color-customized through the Auth Server’s ability to apply enterprise branding packages on login.

Admin

Version

Admin has been proven to be compatible with Rivet v4. It is recommended that it remain on Rivet v4 unless a major update is performed (e.g. an Angular version update) that would require a Rivet update. It is also possible that future patch updates be applied if bugs caused by Rivet are found.

Branding

Admin follows the branding variable standards as established in Rivet v4, and can be color-customized through the Auth Server’s ability to apply enterprise branding packages on login.

Client Portal

Version

Client Portal was built using an the unofficial ‘v2’ of Rivet that was built as part of the Client Portal software. It is not possible to upgrade the version of Rivet without a significant re-design of the UI and a significant upgrade from AngularJS.

Branding

Client Portal uses its own set of branding variables seperate from Rivet. Follow this documentation for instructions on how to override Client Portal branding.

Profiles

There is no compatibility between Rivet and Profiles, and there is no future plan to utilize Rivet in Profiles
