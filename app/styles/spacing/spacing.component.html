Spacing

Spacing is defined through reusable CSS variables to help narrow down the dimensions we use for spacing, thereby speeding up the design review process.

Spacing variables

Implementation

To use spacing variables, set the margin/padding of a component or element to one of the variables. In order to reference the variables, make sure to use var( x ); with 'x' being one of the variables listed below.
Example scss: .container { margin-bottom: var(--rvt-spacing-lg); }

Size Variable Default Value Example
{{v.name}} {{v.var}} {{v.px}}

Spacing classes

Rivet also generates spacing classes to apply directly to components and elements to quickly add margins or padding.

Implementation

To use the classes, use the prefixes listed below (example mt) for your desired outcome, then add a hyphen - followed by the spacing size you need (example xxl). These values are listed in the table above.
Example HTML: class="mt-xxl"
