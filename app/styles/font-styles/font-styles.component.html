Font Styles

Font mixins

Mixins contain multiple set font styles (font weight, font family, and font size) within the include.
Example mixin:
@mixin rvt-font-h {
font-weight: var(--rvt-font-weight-regular);
font-size: var(--rvt-font-size-h);
font-family: var(--rvt-font-family);
}

All mixins (except font-g & font-h) also include logic to resize at smaller screen sizes. This helps maintain hierarchy while allowing more content to fit on the page with smaller screens.

Implementation

To utilize the SCSS mixins that contain our font rules you must import the stylesheet that contains the mixins by doing @import 'typeStyles';

Within the css, use @include (mixin-name). The list of mixins and their sizes are included below. All mixins are set to a regular font weight.

Font family

Rivet uses Nunito Sans as the only font-family for all content. Nunito Sans is imported in the Rivet package so there is no need to import it within individual projects.

Implementation

To set the font family in the css, use font-family: var(--rvt-font-family);

Font classes

While Rivet does support applying the above font mixins as classes directly to an element or component, it is recommended to include the mixin with the css instead. This allows the text to maintain the ability for live style changing (such as increasing text size).

Implementation

Font weights

Font weights are used to help establish hierarchy within a page.

Implementation

Currently Rivet has 3 font weights to use: light, regular, and bold.
To set the font weight of text, use font-weight: var(--rvt-font-weight-light);

Examples

Light text (200) font-weight: var(--rvt-font-weight-light);

Regular text (400) font-weight: var(--rvt-font-weight-regular);

Bold text (700) font-weight: var(--rvt-font-weight-bold);

Font sizes

Implementation

To set text to a specific font size, use Rivet font size variables by using font-size: var(--rvt-font-size-a);.

Use

For use within products, sizes display-1 through size-d will be commonly used for headers. Sizes e, f, and g will be used for body copy. Size-h should be reserved for labels and captions. Consult with your designer to see which size should be used.
