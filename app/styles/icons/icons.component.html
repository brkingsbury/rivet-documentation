Atypical direct CSS usage

!IMPORTANT: starting in v0.5.3 icons referenced in css via unicode characters will no longer be supported. To add an icon through css use:

.element:before ｛ @extend .inner-rivet-icon-name !optional; ｝

  • :before it is required that the icon be added as a pseudo-element
  • @extend will reference the mixin from the Rivet icon font stylesheet
  • inner- is a required prefix before the name of the icon
  • rivet-icon-name is the standard name of the icon you want to use from the list below.
